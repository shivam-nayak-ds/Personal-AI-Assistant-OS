## Interview Question Template:

```markdown
### Question: Why did you choose X over Y?

**Short Answer:**
[2-3 sentence summary]

**Detailed Answer:**
[Technical explanation with reasoning]

**Tradeoffs:**
- Pros of X
- Cons of X
- When Y would be better

**Your Implementation:**
[Link to code or explain how you implemented it]

**Follow-up Questions:**
- How would you scale X?
- What if X fails?
- How do you monitor X?
```
